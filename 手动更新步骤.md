# 手动更新步骤

由于 token 权限限制，以下项目需要你手动在 GitHub 网页上更新：

## 1. 更新仓库描述

1. 访问：https://github.com/jiuyaozhuce/lovelace-touchpad-card/settings
2. 在 "Description" 字段中输入：
   ```
   Home Assistant 触控遥控器卡片 - 支持滑动、点击、按钮控制等多种操作模式
   ```
3. 点击 "Save changes"

## 2. 更新 Release 描述（可选）

1. 访问：https://github.com/jiuyaozhuce/lovelace-touchpad-card/releases/tag/v1.1.0
2. 点击 "Edit release"
3. 在 "Describe this release" 中输入：

```
## 更新内容

- ✨ 新增 **按钮控制模式**（button mode）
- 现在可以直接触发按钮实体（button.xxx）来控制设备
- 新增按钮实体配置选项（button_up、button_down 等）

## 使用方法

设置 `control_mode: button`，然后配置按钮实体：

```yaml
type: custom:touchpad-card
control_mode: button
button_up: button.tv_up
button_down: button.tv_down
button_ok: button.tv_ok
```
```

4. 点击 "Update release"
